# The Heart and Integrated UI - btc-core

**btc-core is the heart of all btc scripts; it analyzes your framework and makes everything work as it should!**

This document details the functionality of the NUI system integrated into `btc-core`. It is designed to be a robust and customizable foundation for all other scripts on your server, offering a wide range of UI features, including a menu with two distinct visual themes and a standalone progress bar.
